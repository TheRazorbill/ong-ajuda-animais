# Paws & Hearts Rescue - Plataforma Web para ONG

Este é o repositório oficial da plataforma web para a ONG "Paws & Hearts Rescue", um projeto focado em ajudar no resgate, reabilitação e adoção de animais em situação de vulnerabilidade.

## Status do Projeto

**Entrega I (HTML):** Concluída ✔️
* Estrutura semântica para 3 páginas (Início, Projetos, Cadastro).
* Uso de hierarquia de títulos e formulário complexo.
* Estrutura de pastas inicial definida.

**Entrega II (CSS):** Concluída ✔️
* Design system com variáveis CSS.
* Leiautes responsivos com Flexbox e Grid.
* Componentes de interface estilizados (cards, botões, formulários).

**Entrega III (JavaScript):** A fazer.
